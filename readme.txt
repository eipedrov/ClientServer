Servidor: execute o arquivo "main.py" para deixar o servidor aberto a conexoes
Cliente: execute o "User.py" quantas vezes forem necessárias para simular o usuario do chat
e mandar mensagens para o servidor. As mensagens serão recebidas pelo servidor e listadas
na ordem que chegam, em atualizações futuras o servidor poderá mandar uma resposta de atualização
para todos os clientes, assim atualizando em tempo real.
